# vue-socketio-private-messaging-tutorial

## Server setup
```
1) cd server
2) npm install
3) node index.js
```

## Client setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
